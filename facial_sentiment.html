Facial Sentiment Analysis

Can we recognize Sentiment from video?

Detecting a users opinion has always been a relevant area of study due to the feedback that it provides to a given brand or service provider because it can help improve dramatically, after all client satisfaction depends on the client being satisfied.

Domain Background

In order to improve the client satisfaction and measuring the impact of a given input in the public, there have been several approaches. One of the most common and used is sentiment analysis via text. With this, you can train a AI model that can tell whether a comment is either positive or negative, which proved to be really useful when analyzing the costumer reviews and user comments in general. There have been several attempt to mimic this analysis in order to determine the sentiment in images, one example is:
http://www.ifp.illinois.edu/~jyang29/papers/AAAI15-sentiment.pdf
where a set of famous images where used to detect the sentiment that the images depicted.



Problem Statement

he problem is simple, given an image, we want to detect which emotion is depicted. More specifically, from a pool of 6 possible emotions, we will choose the most probable one as the emotion being expressed in a picture of a person. This process will later be extrapolated in order to detect emotions live using video from a webcam.



Datasets and Inputs

We will use the Cohn-Kanade AU-Coded Expression Database (http://www.pitt.edu/~emotion/ck-spread.htm) which contains a set of images focusing the faces of several people gesturing 6 differentemotions: 0=neutral, 1=anger, 2=contempt, 3=disgust, 4=fear, 5=happy, 6=sadness, 7=surprise.In this dataset, we can observe that each image is labeled, also, the faces are not always facing the camera. Furthermore, there are some cases where the person in the picture is not facing forward but has a slight head rotation.



Solution to the problem

As it is an image classification problem, We proposed as a solution, a trained deep convolution neural network to learn from the images to then classify them accordingly. This is an approach that has proven to be a state of the art approach and is the most accurate method of solving image classification problems as shown in: https://papers.nips.cc/paper/4824-imagenet-classification-with-deep-convolutional-neural-networks.pdf.

The paper above is one of the most cited, relevant and influential in the matter where a convnet is exemplified using what is now known as an Alex architecture or AlexNet, which is made of 5 conv layers, max-pooling layers, dropout layers and 3 fully connected layers. A similar architecture was adapted to the problem in hand in order to achieve optimal results.
